---
title: SQLite
---

#### command line tool:

```
sqlite3
```

Ex:

```
sqllite3 mydb.db
```

#### change to line mode

```
.mode line
```

#### get database information

```
select * from sqlite_master;
```

#### show tables

```
sqlite> .tables
```

#### show table schema (columns, datatypes, etc.)

```
sqlite> .schema TABLE_NAME
```
